Property Hub Case Study

Problem

Landlords were managing properties, tenants, and rent records across spreadsheets and chat tools, which caused missed updates and inconsistent data.

Approach

I designed a MERN architecture with role-based flows, REST APIs for core entities, and a React dashboard that keeps listing and rent data synchronized in real time.

Outcome

The platform provides one source of truth for property operations and reduces manual follow-ups by centralizing rent tracking, tenant status, and listing updates.
